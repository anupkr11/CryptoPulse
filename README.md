# Cryptocurrency Dashboard

A professional cryptocurrency dashboard built using **React**, **Redux**, and **TailwindCSS**.  
The application provides real-time cryptocurrency market data, currency conversion, and interactive visualizations powered by the **CoinGecko API**.
